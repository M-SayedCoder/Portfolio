# Portfolio Website

## Required Assets

To complete the portfolio website, you need to add the following files:

### Images (in assets/images/)
1. **profile.jpg** - Your main profile photo (recommended: 400x400px, circular)
2. **about.jpg** - About section photo (recommended: 400x500px)
3. **project1.jpg** - Project 1 screenshot (recommended: 400x250px)
4. **project2.jpg** - Project 2 screenshot (recommended: 400x250px)
5. **project3.jpg** - Project 3 screenshot (recommended: 400x250px)

### Documents (in assets/)
1. **cv.pdf** - Your updated CV/resume

## Features Included

✅ Navbar (sticky, transparent→dark on scroll, mobile collapse)
✅ Hero Section (with typing effect, two buttons)
✅ About Section (photo, bio, animated stats)
✅ Skills Section (3 columns with animated progress bars)
✅ Experience Section (vertical timeline)
✅ Services Section (4 cards with hover effects)
✅ Projects Section (with Bootstrap Modals - Case Study style)
✅ Achievements Section
✅ Testimonials Section (Bootstrap Carousel)
✅ Contact Section (form + social links)
✅ Footer

### JavaScript Effects
✅ Typing Effect
✅ Smooth Scroll
✅ Navbar change on scroll
✅ Scroll reveal animation
✅ Back to top button
✅ Stats counter animation
✅ Skills progress bar animation

### Responsive Design
✅ Mobile-optimized (below 768px)
✅ Tablet support (768px - 991px)
✅ Desktop full support

## Color Scheme
- Background Dark: #0D1B2A
- Card Background: #1B263B
- Primary Orange: #FF6F00
- Primary Hover: #ff8c1a
- Text Main: #E0E1DD
- Text Muted: #A9B4C2

## How to View
1. Open index.html in your browser
2. Or use a local server (recommended for best experience)

## Customization
- Update personal information in index.html
- Replace placeholder images with your own
- Update project details in the modals
- Modify contact form endpoint as needed
